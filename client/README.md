# Client

## Project setup
```
# yarn
yarn

# npm
npm install
```

### Compiles and hot-reloads for development
```
# yarn
yarn dev

# npm
npm run dev
```
After you run  `yarn dev`, go to `http://localhost:3000/` to see the site live on your local!

### Compiles and minifies for production

```
# yarn
yarn build

# npm
npm run build
```
